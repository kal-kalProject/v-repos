# official-doc/

Documentación **oficial** de la plataforma Vortech.

Contenido de este directorio:

- **Descripción de plataforma** — qué es Vortech, sus primitivas, sus capas, sus principios.
- **Pitch / narrativa pública** — cómo se presenta la plataforma a externos.
- **Conceptos y vocabulario canónico** — la fuente de verdad léxica.

Lo que **no** vive aquí:

- Decisiones de stack, de unificación, de cierre arquitectónico (→ [`inventory/_analysis-preview/`](../inventory/_analysis-preview/)).
- Planes de implementación, migración, roadmap (→ `v-mono/analysis/` cuando exista).
- Inventario técnico de repos (→ [`inventory/`](../inventory/)).
- Análisis operativo, comparativas, triage (→ `inventory/_analysis-preview/`).
